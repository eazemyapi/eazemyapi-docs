# Creating Database Tables

Tables are the foundation of your backend.

Each table represents a collection of data.

Example tables:

Users
Products
Orders
Messages

## How to Create a Table

1. Open your project dashboard
2. Click **Create Table**
3. Enter table name
4. Add fields

Example fields for Users table:

* id (Primary Key)
* name (Text)
* email (Text)
* password (Text)
* created_at (Datetime)

After saving, EazeMyAPI automatically prepares APIs for this table.

---

## Best Practices

* Use clear table names
* Keep fields simple
* Avoid unnecessary columns
* Plan relationships early
